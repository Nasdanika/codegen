Base class for wizard page content. Wizard implementations shall provide concrete subclasses of this class.

shallShow : EBoolean - Returns true (default) if the page shall be shown.

Returns true (default) if the page shall be shown. Override to conditionally skip pages. For example, if the user doesn’t check, say “Connect to database” checkbox, when then there is no point in showing “Database Connection Configuration” page.

Type EBoolean
Cardinality 0..1
Exceptions
    Unique

    validate : EBoolean - Validates the content.

    Validates the content. This method is used by isPageComplete(), which is turn is used by canFlipToNextPage() and canFinish(). isPageComplete() returns true if vaildation yields no error level diagnostics.
